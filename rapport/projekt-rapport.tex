\section{Background}
The objective in this project is to create an application where the project members knowledge in network programming is utilized in as great an extent as possible. 
The project members' love for arcade- and retro-esque games in combination with the desire to create a kickass network application has resulted in a multiplayer game with gameplay influences from Space Invaders and the look and feel of Asteroids.     
 
The plan is to have up to four players to be able to view eachother in real time while playing and also be able to interact with eachother. All this done through network communication.
